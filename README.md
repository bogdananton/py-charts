# Package and module

```bash
# PIL (python image library) - used for saving images
pip install pillow
pip install matplotlib numpy uuid
```

Run sample

```bash
python sample.py
```

### Testing

Install test dependencies

```bash
pip install pytest
```

Run tests

```bash
python -m pytest test
```
